# 管理员命令

## 如何使用

| 命令                | 描述                                 |
| ----------------- | ---------------------------------- |
| /bungeehub help   | 查看帮助                               |
| /bungeehub check  | 检查更新                               |
| /bungeehub reload | 重载配置文件 (config.yml 和 messages.yml) |

{% hint style="warning" %}
要使用这些命令，需要 `bungeehub.admin` 权限。
{% endhint %}
